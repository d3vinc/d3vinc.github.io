## How to use

```
chenyang@chenyang-pc:/tmp$ cat sequenceDiagram.txt
@startuml
Alice -> Bob: test
@enduml

chenyang@chenyang-pc:/tmp$ java -jar ~/Downloads/plantuml.jar sequenceDiagram.txt

chenyang@chenyang-pc:/tmp$ eog sequenceDiagram.png
```
